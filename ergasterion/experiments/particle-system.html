PARTICLE SYSTEM

$ particle_system.exe

An interactive particle system exploring emergent behavior through physics simulation. Each particle responds to mouse position with attraction and repulsion forces, creating organic, flowing patterns in real-time.

Interactive Controls
Move your mouse over the canvas to control particle behavior. Particles accelerate towards or away from the cursor based on proximity. Watch how simple attraction/repulsion rules create complex swarm dynamics.
$ Algorithm: Particle Physics 
Particle Motion Algorithm:
  1. Initialize N particles with random positions and velocities
  2. Each frame:
     a. Calculate distance from particle to mouse cursor
     b. If distance < attractionRadius: apply attraction force
     c. If distance < repulsionRadius: apply repulsion force
     d. Update velocity using F = ma (Newton's Second Law)
     e. Apply drag/friction to velocity
     f. Update position: position += velocity
     g. Wrap particles around screen edges
     h. Draw particle with age-based opacity
Key Physics:
  - Force magnitude decreases with distance squared (inverse square law)
  - Drag coefficient dampens oscillations (natural friction)
  - Color changes based on particle age (lifecycle visualization)
  - Trail effect creates motion blur perception

// PARTICLE SYSTEM with p5.js
// Demonstrates emergent behavior from simple physics rules
let particles = [];
const PARTICLE_COUNT = 100;
const ATTRACTION_RADIUS = 150;
const REPULSION_RADIUS = 50;
const MAX_FORCE = 0.5;
const DRAG = 0.95;
function setup() {
  let container = document.getElementById('sketch-container');
  let w = container.offsetWidth;
  let h = 500;
  let sketchCanvas = createCanvas(w, h);
  sketchCanvas.parent('sketch-container');
  // Initialize particles
  for (let i = 0; i < PARTICLE_COUNT; i++) {
    particles.push(new Particle(
      random(w),
      random(h)
    ));
  }
}
function draw() {
  background(0, 0, 0, 25);
  // Update and draw particles
  for (let particle of particles) {
    particle.update(mouseX, mouseY);
    particle.display();
  }
  // Draw cursor indicator
  stroke(191, 255, 0, 100);
  strokeWeight(2);
  noFill();
  circle(mouseX, mouseY, ATTRACTION_RADIUS * 2);
  circle(mouseX, mouseY, REPULSION_RADIUS * 2);
}
class Particle {
  constructor(x, y) {
    this.pos = createVector(x, y);
    this.vel = p5.Vector.random2D().mult(2);
    this.acc = createVector(0, 0);
    this.age = 0;
  }
  update(targetX, targetY) {
    let target = createVector(targetX, targetY);
    let distance = dist(
      this.pos.x, this.pos.y,
      targetX, targetY
    );
    // Attraction force
    if (distance < ATTRACTION_RADIUS && distance > REPULSION_RADIUS) {
      let direction = p5.Vector.sub(target, this.pos);
      direction.normalize();
      let force = direction.mult(MAX_FORCE * 0.3);
      this.acc.add(force);
    }
    // Repulsion force
    if (distance < REPULSION_RADIUS) {
      let direction = p5.Vector.sub(this.pos, target);
      direction.normalize();
      let force = direction.mult(MAX_FORCE);
      this.acc.add(force);
    }
    // Apply physics
    this.vel.add(this.acc);
    this.vel.mult(DRAG);
    this.pos.add(this.vel);
    this.acc.mult(0);
    // Wrap around edges
    this.pos.x = (this.pos.x + width) % width;
    this.pos.y = (this.pos.y + height) % height;
    this.age++;
  }
  display() {
    let opacity = map(this.age, 0, 300, 255, 0);
    let speed = this.vel.mag();
    let hueValue = map(speed, 0, 5, 120, 50);
    colorMode(HSB);
    stroke(hueValue, 100, 100, opacity);
    strokeWeight(2);
    point(this.pos.x, this.pos.y);
    colorMode(RGB);
  }
}
$ Tunable Parameters 
PARTICLE_COUNT = 100
  Increase for more visible swarms, decrease for cleaner animation
ATTRACTION_RADIUS = 150
  Distance at which particles begin accelerating toward cursor
  Larger values = particles respond from further away
REPULSION_RADIUS = 50
  Inner radius where particles flee from cursor
  Creates the "avoid" zone around cursor
MAX_FORCE = 0.5
  Maximum acceleration magnitude per frame
  Higher values = snappier, more responsive movement
DRAG = 0.95
  Velocity dampening factor (0-1)
  Lower = more friction, smoother motion
  Higher = less friction, more momentum
Emergent Behavior
This system demonstrates how complex, organic-looking behavior emerges from simple rules. No particle "knows" about others—they only respond to the mouse cursor and physics. The collective swarm behavior appears intelligent yet requires no explicit coordination. This is the basis of flocking algorithms used in games, animation, and robotics.
